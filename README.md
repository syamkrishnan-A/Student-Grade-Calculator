# Student-Grade-Calculator


## Project Description
This is a simple Python-based application that takes student marks as input and calculates the grade based on predefined grading criteria. It also displays encouraging messages based on performance.

## Features
- Input validation (marks between 0–100)
- Grade calculation using if-elif-else
- Encouraging messages for each grade
- User-friendly output format
- Continuous input handling using while loop

## 🛠️ Technologies Used
- Python

## 📊 Grading System
- A: 90–100
- B: 80–89
- C: 70–79
- D: 60–69
- F: 0–59
